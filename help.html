naija flappy bird game documentation

the niger flappy bird game, is an intresting game to play... it consists of a player which is a bird, the player has to dodge the obstacles and planes to survive..

controls

the player can be controlled using the following ways;
  • Keyboard control
  • Game Pad
  • Touch Pad/Screen

Keyboard Control

The player is controlled by pressing the space bar on the keyboard to move the player upward..NOTE: the keyboard control is for the desktop version..

Game Pad Control

The throttle or the game pad axis is used to control the player with the game pad.. the upward axis/throttle moves the player up and the downward throttle moves the player down... NOTE: any of the two game pad axis can be used for the control on the desktop game version.

Touch Screen Control

The throttle or the game pad axis is used to control the player with the game pad.. the upward axis/throttle moves the player up and the downward throttle moves the player down... NOTE: The touch screen control is for the mobile application.

How to play

the game is played by avoiding the obstacles and also the planes, the game gets more difficult as the score increases or as long as you are still playing the game.. avoiding falling to the ground because that's game OVER also, you can take part in the quick game to play any mode of your choice, there are also daily challenges to try out, they are more challenging...

High Score

the game comes with an high score module which keeps track of the user's highscore, challenge your self and beat your highscore...., you can also post your highscore on any social media platform which includes Facebook or Twitter.. the highscore posting plaform,is on the game menu. You can also reset your highcore in the game menu



Night Mode And Day Mode

the game is built with both night mode and day mode, the user can choose either modes to play depending on their preference. you can click the EA button on the game GUI to change or toggle the Night Mode Or Day Mode. Also, in the game you can click the Quick Play button to choose between day mode and night mode... also, the Quick Game mode helps users to select the kind of world they want to play... The Game Sounds are majorly electronic music that are pleasing to the ear... an headphone or ear pod is recommended for better user experience...
